What does SDN stand for?

I didn't think through the name as much as I should have, but SDN stands for "Software Developers' Network".

What inspired SDN?

SDN was created when I realized that a huge part of being a Software Developer is working with people. In many cases, rising Developers aren't in a collegiate or hacker community, so this could serve as that environment - find and post cool projects to work on, reach out to other devs, learn how to work on teams, diversify your portfolio and build interesting things.

What tech does SDN use?

SDN itself was built as a project to learn concepts of complex web applications. It uses:

  • Environment: Ubuntu 15.04
  • Hosted On: Linode
  • Served With: WSGI
  • Framework: Flask
  • Database: MySQL
  • Frontend: Bootstrap
  • ORM: SQLAlchemy
